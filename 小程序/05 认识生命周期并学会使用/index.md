# 微信小程序的生命周期

### 微信小程序每个页面存在生命周期

**onlaunch** ->

* **onload** ----------------->生命周期函数-监听页面加载
* **onReady** ---------------->生命周期函数-监听页面初次渲染
* **onShow** ----------------->生命周期函数-监听页面显示
* **onHide** ------------------->生命周期函数-监听页面隐藏
* **onUnload**---------------->生命周期函数-监听页面卸载
* **onPullDownRefresh**->有二面监听相关处理函数-监听用户下拉动作
* **onReachBottom**------->页面上拉触底时间的处理函数
* **onShareAppMessage**->用户点击右上角分享
