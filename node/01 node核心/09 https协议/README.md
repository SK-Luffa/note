* 加密 ：
  
  * 对称加密： 产生一个密钥，可用其加密，也可以用其解密 
       加密过程：信息+key=[信息，key]
       解密过程；[信息，key] +key=信息
    常用算法； DES、3DES、AES、Blowfish等
  
  * 非对称加密：产生一对密钥，一个用于加密，一个用于解密
      产生密钥对：   公钥 or 私钥
      加密过程：  信息+ （公钥key）=[(公钥key)，信息]
      解密过程：  [(公钥key)，信息]+（私钥key）=信息
    常用算法：RSA、Elgamal、Rabin、D-H、Ecc等
