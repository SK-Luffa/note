 # 模板字符串
 
 ES6之前处理字符串繁琐的两个方面

 1. 多行字符串 

 2. 字符串拼接


 在ES6中，提供了模板字符串的书写，可以非常方便的换行和拼接，要做的，仅仅是将字符串的开始和结尾改为`符号

 #####  如果要在字符串中拼接JS表达式，只需要在模板字符串中使用“${js表达式}”， 并且表达式可以是任意有意义的数据，并且还可以嵌套 例如：
          const a="张三";
          const b=`李四${a}`;
          console.log(b);
          const c=`王五${b}`;
          console.log(c);
##### 并且如果我们需要打印“${}"的话 需要加上一个转义字符 \

          const a="张三";
          const b=`李四\${a}`;
          console.log(b);
