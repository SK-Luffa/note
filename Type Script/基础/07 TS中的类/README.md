**使用访问修饰符**

访问修饰符可以控制类中的某个成员的访问权限（三种）

 - punlic :默认访问修饰符（公开），所有的的代码都可访问
 - private ：私有的，只有在类中可以访问
 - protected:  暂时不说


**属性简写**

如果某个属性，通过构造函数的参数传递，并且不做任何处理的的赋值给该属性。可以进行简写
